# Therapize Releases

Download the latest version of Therapize for your platform.

## Downloads

See the [Releases](https://github.com/andrewtcrooks/therapize-releases/releases) page for all available downloads.

## Installation

- **Windows:** Download and run the `.exe` installer
- **macOS:** Download the `.zip` file, extract it, move to /Applications. See [MAC_TRUST.md](./MAC_TRUST.md) for first-run instructions.
- **Linux:** Download the `.AppImage`, make it executable (`chmod +x`), and run it

## What is Therapize?

Therapize is a local-first desktop app for therapy session documentation. All processing happens on your device - no data leaves your computer.

For source code and development, see the main repository.
